== Installation ==
 
Get this plugin form wordpress plugin area and install it.
 
e.g.
 
1. Upload 'rubsum' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the site dashboard and add your employee of stuff's details and use the shortcode form in the relevent place of your webpage.
4. Go to your website page and I hope it will work for your company's employee or stuffs details.
5. Hope you enjoy this. Thank you for choosing rubsum.
